# civy-static-pages-deploy

[![Deploy to GitHub Pages](https://github.com/York-Solutions-LLC/civy-static-pages-deploy/actions/workflows/deploy.yml/badge.svg)](https://github.com/York-Solutions-LLC/civy-static-pages-deploy/actions/workflows/deploy.yml)

A simple Vue 3 + TypeScript application with automated code quality checks (Prettier, ESLint, TypeScript, Vitest) and deployment to GitHub Pages.

## Deployment

This project is automatically built and deployed to GitHub Pages via GitHub Actions on merge to `main`.

**Live Site:** https://york-solutions-llc.github.io/civy-static-pages-deploy/

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Vitest
- GitHub Actions
- GitHub Pages

## Requirements

- Node.js 18+
- npm

## Commands

```bash
npm install          # Install dependencies
npm run dev          # Start development server
npm run build        # Build for production
npm test             # Run vitest unit tests
npm run type-check   # Check TypeScript types
```
